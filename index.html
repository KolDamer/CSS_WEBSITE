No, Computer Science isn’t just coding. It’s about understanding how computers work, solving problems, and creating technology. Coding is one part of it, but it also includes things like designing software, studying data, and learning about hardware.

Computer Science offers many career options like software development, data analysis, cybersecurity, and AI. You can also work in exciting fields like robotics and cloud computing.
